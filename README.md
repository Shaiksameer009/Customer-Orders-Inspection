# Customer Orders Inspection Assignment

#Objective
This assignment analyzes an e-commerce orders dataset using Python and Pandas.

#Dataset Information
The dataset contains the following columns:
- order_id
- customer_name
- product
- quantity
- unit_price
- order_date
- status

#Tasks Completed

-> Task 1: Load and Preview Dataset
- Loaded dataset using pandas
- Set order_id as index
- Displayed first and last 5 rows

-> Task 2: Structural Inspection
- Checked dataset shape
- Identified missing values
- Converted order_date to datetime format

-> Task 3: Statistical Summary
- Generated summary statistics
- Found median unit_price
- Analyzed distribution skewness

-> Tools Used
- Python
- Pandas
- Google Colab

-> Key Findings
- Dataset contains 20 rows and 6 columns.
- Missing values found in quantity and order_date.
- Median unit_price is ₹2500.
- unit_price is positively skewed due to high maximum value.
